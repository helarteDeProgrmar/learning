# Main

En este fichero estudiamos las partes que se deben ir aprendiendo sobre
sistemas linux. Ademas se ira completando donde encontrar la informacion
pertinente dentro del repositorio para despues ser agiles a la hora de buscar
informacion y resolver problemas.

## Temario

1. Fundamentos de consola
  - `sh` vs `bash` vs `zsh`
  - sistema de archivos
  - i/o, redirecciones y pipes
2. Gestion de archivos y permisos
  - permisos
  - acls
  - enlaces
3. Usuarios y grupos
4. Procesos y rendimiento
5. Redes
6. Servicios y systemd
7. Seguridad
8. Almacenamiento y discos
9. Automatizacion y Scripting
